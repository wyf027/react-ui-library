# ImagePreview 图片预览

图片预览组件。

## 示例

<LivePlayground :code="`
() => {
  return (
    <ImagePreview src='https://images.unsplash.com/photo-1469474968028-56623f02e42e?w=600' />
  )
}
`" />

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| src | 图片地址 | `string` | - |
| alt | 替代文本 | `string` | - |
| width | 宽度 | `number` | - |
| height | 高度 | `number` | - |
