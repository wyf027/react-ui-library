# Button

Primary action control with **solid / outline / ghost** variants and **primary / neutral / danger** colors. Supports `href` for link mode and `loading` for async actions.

- Public API & examples: repository root file `docs/components/button.md` (published on the VitePress docs site).
- A11y: focus ring via `nova-focus-ring`; prefer `type="button"` in forms (default).
