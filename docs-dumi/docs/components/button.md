## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 按钮样式 | `'solid' \| 'outline' \| 'ghost'` | `'solid'` |
| size | 尺寸 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| color | 颜色 | `'primary' \| 'neutral' \| 'danger'` | `'primary'` |
| loading | 加载状态 | `boolean` | `false` |
| disabled | 禁用状态 | `boolean` | `false` |
| icon | 图标 | `ReactNode` | - |
