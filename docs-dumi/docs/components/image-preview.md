## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| src | 图片地址 | `string` | - |
| alt | 替代文本 | `string` | `'preview'` |
| width | 宽度（px） | `number` | `160` |
| height | 高度（px） | `number` | `120` |
