# SplitPane 分栏

左右分栏布局组件。

<code src="./demos/split-pane-basic.tsx"></code>

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| left | 左侧内容 | `ReactNode` | - |
| right | 右侧内容 | `ReactNode` | - |
| ratio | 比例（CSS grid-template-columns 值） | `string` | `'1fr 1fr'` |
