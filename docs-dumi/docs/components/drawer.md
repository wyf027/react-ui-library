## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| open | 是否可见 | `boolean` | - |
| onClose | 关闭回调 | `() => void` | - |
| placement | 抽屉方向 | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` |
| title | 标题 | `ReactNode` | - |
