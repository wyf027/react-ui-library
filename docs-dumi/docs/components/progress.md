## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| percent | 百分比 | `number` | - |
| status | 状态 | `'normal' \| 'success' \| 'exception'` | `'normal'` |
| showInfo | 显示百分比文字 | `boolean` | `true` |
