# Image 图片

图片组件，对齐 Ant Design Image，支持预览和失败占位。

## 示例

<code src="./demos/image-basic.tsx"></code>

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| src | 图片地址 | `string` | - |
| alt | 替代文本 | `string` | `''` |
| fallback | 失败占位图 URL | `string` | `''` |
| placeholder | 加载占位 | `ReactNode` | - |
| preview | 是否支持点击预览 | `boolean` | `true` |
