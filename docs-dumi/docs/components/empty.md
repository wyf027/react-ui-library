## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| description | 描述文字 | `ReactNode` | `'No Data'` |
| image | 自定义图标 | `ReactNode` | - |
| simple | 无边框轻量样式 | `boolean` | `false` |
| imageStyle | 图标容器样式 | `CSSProperties` | - |
