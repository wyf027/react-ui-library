## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 标签项 | `TabItem[]` | - |
| activeKey | 当前激活 | `string` | - |
| defaultActiveKey | 默认激活 | `string` | - |
| onChange | 切换回调 | `(key: string) => void` | - |
