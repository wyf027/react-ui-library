# Popover 气泡卡片

点击触发弹出气泡卡片。

## 示例

<code src="./demos/popover-basic.tsx"></code>

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| trigger | 触发元素 | `ReactNode` | - |
| content | 卡片内容 | `ReactNode` | - |
| open | 是否显示（受控） | `boolean` | - |
| defaultOpen | 默认显示 | `boolean` | `false` |
| onOpen | 打开回调 | `() => void` | - |
| onClose | 关闭回调 | `() => void` | - |
