# Col 列

配合 Row 使用的栅格列组件，12 列栅格。

## 示例

<code src="./demos/col-span.tsx"></code>

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| span | 占据列数（12 栅格） | `number` | `12` |
| offset | 左侧偏移列数 | `number` | `0` |
| flex | 设为 flex 布局项时忽略 `span`/`offset`（对标 Ant Design `flex`） | `string \| number` | - |
| order | CSS `order` | `number` | - |
